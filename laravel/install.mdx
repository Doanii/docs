Deze tutorial beschrijft de installatieprocedure voor een verse Laravel 12 installatie met Laravel Herd. Zorg dat aan de systeemvereisten (PHP, Composer, etc.) voldaan wordt alvorens te beginnen.

## Stap 1: Laravel Herd installeren

1. Navigeer naar de Laravel Herd documentatie:

```
https://herd.laravel.com/docs
```

1. Volg de instructies op de website om Laravel Herd te downloaden en te installeren. Herd zorgt voor een uniforme, lokaal ontwikkelplatform en vereenvoudigt het beheer van jouw Laravel projecten.

## Stap 2: Een nieuw Laravel 12 project aanmaken

1. Open een terminal.

2. Voer het volgende commando uit om een nieuw Laravel 12 project te genereren:

```bash
herd new mijn-laravel-project --version=12
```

Dit commando creëert een nieuwe Laravel 12 installatie in de directory `mijn-laravel-project`.

1. Navigeer naar de projectmap:

```bash
cd mijn-laravel-project
```

## Stap 3: De ontwikkelserver starten

1. Start de lokale ontwikkelserver:

```bash
herd serve
```

1. Open de getoonde URL in een webbrowser om de Laravel installatie te verifiëren.

## Aanvullende instructies

* Controleer of alle vereiste PHP-extensies en Composer correct zijn geïnstalleerd.

* Raadpleeg de Laravel Herd documentatie voor geavanceerde configuratie-opties en eventuele probleemoplossing.

* Houd rekening met toekomstige updates van Laravel en Herd om uw ontwikkelomgeving actueel te houden.
